# 📖 Project Overview

The **Responsive Image Gallery** is a modern web application developed using **HTML5, CSS3, and JavaScript**. The project demonstrates the implementation of an attractive, interactive, and fully responsive image gallery where users can browse images smoothly across different devices.

The gallery provides an enhanced user experience through features like **lightbox image preview, next/previous navigation, hover animations, smooth transitions, responsive layouts, and image categorization**. This project helps strengthen the understanding of frontend web development concepts including DOM manipulation, CSS Flexbox/Grid, responsive design, JavaScript event handling, and UI/UX principles.

This project is created as part of the **CodeAlpha Frontend Development Internship**.

---

# 🎯 Objectives

- 📷 Design a clean and attractive image gallery.
- 🎨 Improve user experience with hover animations and transitions.
- 💡 Implement JavaScript-based image navigation.
- 📱 Build a fully responsive layout for all screen sizes.
- ⚡ Practice DOM manipulation and event handling.
- 🚀 Learn modern frontend development techniques.

---

# ✨ Features

### 🖼️ Responsive Image Gallery
- Displays multiple images in an organized grid layout.
- Automatically adjusts according to screen size.
- Uses CSS Grid/Flexbox for better responsiveness.

---

### 🔍 Lightbox Image Preview
- Clicking an image opens it in a fullscreen lightbox.
- Background becomes dark to focus on the selected image.
- Provides an immersive viewing experience.

---

### ⬅️➡️ Image Navigation
- Previous and Next buttons allow users to browse images without closing the lightbox.
- Smooth switching between gallery images.
- Keyboard navigation can also be added for better accessibility.

---

### 🎨 Hover Effects
- Image zoom animation on hover.
- Shadow effects for modern UI.
- Cursor changes to indicate clickable images.

---

### ✨ Smooth Animations
- CSS transitions provide elegant animations.
- Lightbox opening and closing effects.
- Image scaling and fading animations.

---

### 📱 Fully Responsive Design
- Optimized for:
  - 💻 Desktop
  - 💼 Laptop
  - 📱 Mobile
  - 📟 Tablet

- Responsive layout using media queries.

---

### 🗂️ Bonus Feature – Image Categories
Users can filter images based on categories such as:

- 🌄 Nature
- 🏙️ City
- 🐶 Animals

Filtering is implemented using JavaScript without reloading the page.
